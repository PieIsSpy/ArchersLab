# Archers Lab 2

Installing:
1. Clone the repo or install the zip file and unzip it
2. Run `npm install` inside `client` and `server` folders
3. Run `npm install` inside the project's parent folder
4. Create your own `.env` file inside the `server` and `client` folder
   - look at `/server/.env.example` for instructions
   - look at `/client/.env.example` for instructions

Running: \
Run `npm run dev` inside the project parent folder
